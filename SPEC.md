# PetFriends - Telegram Mini App (аналог Widgetable)

## Concept & Vision

Уютное мини-приложение для выращивания виртуальных питомцев в Telegram. Минималистичный, но эмоциональный опыт — питомец живой, реагирует на действия, имеет характер. Главный экран чистый, питомец в центре внимания. Весь функционал доступен через интуитивные жесты и меню.

## Design Language

### Aesthetic Direction
Мягкий, дружелюбный дизайн с rounded элементами. Вдохновение — Tamagotchi meets modern UI. Пастельные тона с яркими акцентами.

### Color Palette
- Primary: `#6C5CE7` (фиолетовый)
- Secondary: `#A29BFE` (светло-фиолетовый)
- Accent: `#FD79A8` (розовый)
- Background: `#F8F9FA` (светло-серый)
- Surface: `#FFFFFF` (белый)
- Text Primary: `#2D3436`
- Text Secondary: `#636E72`
- Success: `#00B894`
- Warning: `#FDCB6E`
- Danger: `#E17055`

### Typography
- Primary Font: `Inter` (Google Fonts)
- Emoji Font: системные emoji

### Spatial System
- Base unit: 8px
- Border radius: 16px (cards), 24px (buttons), 50% (avatars)
- Shadows: soft, layered (`0 4px 20px rgba(0,0,0,0.08)`)

### Motion Philosophy
- Pet animations: CSS keyframes, 60fps
- UI transitions: 200ms ease-out
- Micro-interactions: bounce на тапах
- State changes: плавные fade/scale

## Features & Interactions

### Система питомцев

**Типы питомцев:**
- 🐱 Кот - независимый, любит спать
- 🐶 Собака - энергичный, любит играть
- 🐰 Кролик - тихий, любит есть
- 🦊 Лиса - хитрая, любит исследовать
- 🐻 Медведь - сильный, медленно растёт
- 🐼 Панда - ленивый, много ест
- 🐨 Коала - сонный, мало требует
- 🐸 Лягушка - весёлая, прыгает

**Характеры (влияют на поведение и скорость прокачки):**
- 😊 Активный: быстрее растёт, больше ест
- 😴 Спокойный: медленнее растёт, меньше требует
- ⭐ Любопытный: открывает секреты, любит исследование

**Статы питомца:**
- ❤️ HP (здоровье): 0-100, восстанавливается едой
- 🍎 Hunger (голод): 0-100, уменьшается со временем
- 😊 Mood (настроение): 0-100, повышается игрой
- ⭐ XP (опыт): 0-100, растёт от действий
- 🌟 Level (уровень): 1-∞

### Уход за питомцем (улучшенный геймплей)

**Действия:**
1. **🍽️ Кормить** - пополняет Hunger, восстанавливает немного HP
   - Разные виды еды дают разный эффект
   - 🍎 Яблоко: +20 Hunger, +5 HP
   - 🥛 Молоко: +30 Hunger, +10 HP
   - 🍰 Торт: +50 Hunger, +20 HP, +10 Mood
   - 🦐 Деликатес: +70 Hunger, +30 HP, +20 Mood

2. **🎮 Играть** - пополняет Mood, небольшой расход Hunger
   - Мини-игры для прокачки связи
   - 👋 Погладить: +15 Mood
   - 🎾 Мячик: +25 Mood, -10 Hunger
   - 🧸 Прятки: +35 Mood, -15 Hunger
   - 🏆 Приключение: +50 Mood, -30 Hunger, +10 XP

3. **💊 Лечить** - восстанавливает HP, требует предмет
   - 🩹 Аптечка: +30 HP
   - 💊 Лекарство: +50 HP
   - 🏥 Визит к врачу: +100 HP (требует монеты)

4. **🚿 Ухаживать** - бонусные эффекты
   - 🛁 Купание: +10 Mood, +5 HP
   - 🧹 Расчёсывание: +15 Mood
   - 💤 Отдых: восстанавливает HP быстрее

**Система событий:**
- Питомец периодически генерирует случайные события
- 💤 "Митти хочет спать..." (-5 HP если игнорировать)
- 🍽️ "Митти голодный..." (-10 HP если игнорировать)
- 🎉 "Митти счастлив!" (+5 Mood)
- 🌈 "Митти нашёл что-то!" (+5 XP)
- 😢 "Митти грустит..." (-10 Mood если игнорировать)
- ⚡ "Митти болеет!" (-20 HP если игнорировать)

**События-миниигры:**
- Поймай предмет! - тап появляющемуся объекту
- Угадай настроение! - выбери правильный ответ
- Найди спрятанный предмет! - поиск по экрану

**Прогрессия:**
- Level up каждые 100 XP
- Новые способности/анимации на каждом уровне
- Эволюция питомца на 5, 10, 15 уровнях

### Система предметов (Inventory)
- Рюкзак с предметами для ухода
- Покупка за монеты (soft currency)
- Синтез предметов из компонентов

## Component Inventory

### PetCard (карточка питомца)
- Default: показывает имя, уровень, иконка, краткий статус
- Hover: подсветка, появление кнопок действий
- Active: выбранный питомец
- Disabled: серый, недоступен

### ActionButton (кнопка действия)
- Default: rounded, цвет зависит от типа
- Hover: scale 1.05, glow
- Active: scale 0.95, darker
- Loading: spinner, disabled
- Disabled: opacity 0.5

### StatBar (полоска статы)
- Зелёная (>50%), жёлтая (25-50%), красная (<25%)
- Анимация изменения значения
- Иконка + название + значение

### EventCard (карточка события)
- Timestamp, иконка типа, описание
- Swipe для удаления/отметить прочитанным

### PetAvatar (аватар питомца)
- ASCII art с анимацией idle
- Эмоции меняются по состоянию
- Разные размеры для разных контекстов

## Technical Approach

### Stack
- Pure HTML5 + CSS3 + Vanilla JavaScript
- Telegram Web App SDK
- LocalStorage для persistence
- No backend (client-side only for MVP)

### Architecture
```
/
├── index.html          # Entry point
├── styles/
│   └── main.css        # All styles
├── js/
│   ├── app.js          # Main app controller
│   ├── pet.js          # Pet class & logic
│   ├── ui.js           # UI components
│   ├── storage.js      # LocalStorage wrapper
│   ├── events.js       # Event system
│   └── telegram.js     # Telegram API wrapper
└── assets/
    └── (inline SVGs)
```

### Data Model
```javascript
Pet {
  id: string,
  name: string,
  type: 'cat'|'dog'|'rabbit'|'fox'|'bear'|'panda'|'koala'|'frog',
  personality: 'active'|'calm'|'curious',
  stats: {
    hp: number (0-100),
    hunger: number (0-100),
    mood: number (0-100),
    xp: number (0-100),
    level: number (1-∞)
  },
  inventory: Item[],
  createdAt: timestamp,
  lastInteraction: timestamp
}

User {
  coins: number,
  pets: Pet[],
  activePetId: string|null,
  achievements: string[]
}
```

### State Management
- Single source of truth в localStorage
- Реактивное обновление UI через custom events
- Debounce сохранения (300ms)

### Telegram Integration
- Haptic feedback на действия
- Telegram theme colors adaptation
- BackButton navigation
- Иконка и имя приложения
